# Online-novels
这款微信小程序网络小说平台，是书虫们的掌上图书馆。首页分类超全，玄幻、言情、悬疑、都市等应有尽有，还设有热门榜单，快速锁定人气好书。搜索功能强大，输入书名或作者，瞬间定位目标作品。章节页面简洁清爽，阅读体验舒适，可自由调节字体大小、背景颜色。支持离线缓存，无网络也能畅读。阅读过程中，系统依据喜好推荐相似书籍。无需下载，随时随地打开微信，一键开启精彩阅读之旅 。
